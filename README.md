# arch

Minimal scaffold for an architecture portfolio site (plain HTML/CSS/JS).

Files added/updated:
- `index.html` — base HTML linking `styles.css` and `script.js`
- `styles.css` — minimal base stylesheet
- `script.js` — minimal JS entry
- `assets/` — static assets folder (placeholder `.gitkeep`)

Preview locally:
- Open `index.html` in your browser or serve with `python -m http.server`.

Push to GitHub:
1. git add .
2. git commit -m "Scaffold base portfolio files"
3. git push origin main
